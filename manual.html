1. Introduction

EduCore Pro is a comprehensive educational management system designed to streamline school operations and enhance communication between administrators, teachers, and students. This platform provides role-based access with specialized dashboards and features tailored for each user type.

✓ Dark Theme Interface

2. Getting Started

System Requirements:

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • JavaScript enabled
  • Active internet connection
  • No additional software installation needed

Accessing the System:

  1. Open your web browser
  2. Navigate to the EduCore Pro login page (index.html)
  3. Enter your credentials (email and password)
  4. You will be automatically directed to your role-specific dashboard

8. Troubleshooting

Common Issues and Solutions:

❌ Login Problems

Issue: "Cannot log in, page redirects to login"
  • Clear browser cookies and cache, try again
  • Verify correct email format (e.g., student@school.edu)
  • Confirm password is correct (case-sensitive)
Issue: "Wrong credentials message"
  • Ensure email matches exactly (check for spaces)
  • Use demo credentials for testing
  • Contact admin if credentials are incorrect

❌ Page Display Issues

Issue: "Sidebar not showing or styling looks wrong"
  • Refresh the page (Ctrl+F5 or Cmd+Shift+R)
  • Clear browser cache and reload
  • Try a different browser
Issue: "Data not loading or showing outdated information"
  • Perform a hard refresh of the page
  • Clear browser localStorage
  • Log out and log back in

Browser Compatibility:

Browser Version Support
Chrome Latest ✓ Fully Supported
Firefox Latest ✓ Fully Supported
Safari Latest ✓ Fully Supported
Edge Latest ✓ Fully Supported

Clearing Browser Storage:

  1. Open browser Developer Tools (F12)
  2. Go to Application/Storage tab
  3. Find "localStorage"
  4. Remove entries: edu_auth, edu_students
  5. Refresh the page

9. Frequently Asked Questions (FAQ)

Q: Is my data saved permanently?

A: Data is stored in your browser's local storage. Clearing browser data will remove it. In production, data would be stored in a secure database.

Q: Can I access from multiple devices?

A: Currently, each device has separate storage. Future versions will include cloud synchronization.

Q: What if I forget my password?

A: Use demo credentials for testing, or contact your system administrator.

Q: Can students edit their own grades?

A: No, only teachers and admins can modify grades. Student view is read-only for security.

Q: How often is attendance recorded?

A: Teachers record attendance for each class session. Records are timestamped automatically.

Q: Can I print reports?

A: Yes, use your browser's print function (Ctrl+P or Cmd+P) to print any page as PDF.

Q: Is the system secure?

A: This is a demo system with local storage. Production version includes SSL encryption, secure authentication, and database security.

Q: What browser should I use?

A: Any modern browser works - Chrome, Firefox, Safari, or Edge. Internet Explorer is not supported.
